# Projektaufgaben IST

Hier finden Sie Informationen zu den Abschlussprojekten im Modul "Industrielle Steuerungstechnik", die Sie in Gruppenarbeit durchführen sollen. Die Infos zu Ihrem Projekt finden Sie im entsprechenden Ordner. Dort befindet sich eine **Aufgabenstellung** und einen **Prüfungsbogen**, auf welchem Sie die schriftliche Ausarbeitung machen sollen. Zusätzlich gibt es zu jedem Projekt einen Ausgangszustand der Steuerung, welcher in mehreren **tpzip-Files** zur Verfügung gestellt ist.
